Lorsqu'elle est activée, le plugin va ajouter un fichier "pdf" sur l'exécution du test sur le résultat de l'exécution du cas d'utilisation dans TestLink. Le fichier "pdf" doit avoir le nom de la valeur définie dans le cas KeyCustomField utilisation de Testlink respectif et le changement "#" en "_", par exemple: si KeyCustomField est es.s2o.test.PlatjaTest # testIsSunny le fichier doit être nommé: es.s2o.test.PlatjaTest_testIsSunny.pdf Exemple d'utilizació del txt peut stocker des enregistrements de test.
