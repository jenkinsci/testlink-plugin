Quando activo, o plugin irá anexar um arquivo "pdf" sobre a execução to teste, o resultado da execução do caso de uso em TestLink. O arquivo "pdf" deve ter o nome do valor definido no caso KeyCustomField respectivo no uso Textlink, mas trocando de "#" para "_". Por exemplo: se KeyCustomField é es.s2o.test.PlatjaTest#testIsSunny o nome do arquivo tem que ser es.s2o.test.PlatjaTest_testIsSunny.pdf. Este arquivo "pdf" pode ser utilizado para armazenar alguns registros da execução de teste.
