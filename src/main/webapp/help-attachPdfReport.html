When enabled, the plugin will attach a "pdf" file containing the run tests with the execution results of the use cases in Testlink. The "pdf" file has to be the same name as the value defined in the use case KeyCustomField respective in the Testlink, but changing "#" to "_". E.g. if KeyCustomFiels is es.s2o.test.PlatjaTest#testIsSunny the file naming is es.s2o.test.PlatjaTest_testIsSunny.pdf. Possibles usages of this "pdf" is that it can stored some details of the execution of the test.
